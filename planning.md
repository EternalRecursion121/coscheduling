# Library Coworking Scheduling App

## What is this?
An app to schedule a time to work with friends

## V1 Features
- Sign in via email
- Calendar Sync
- Calendar to create events
    - specify location 
    - invite people (all friends, specific people, all those available)
- Add friends by email
- Autocomplete search
